# Frontend - Checklist App

## Instalación local

1. Entra a la carpeta `frontend`.
2. Instala dependencias:
   ```bash
   npm install
   ```
3. Inicia la app:
   ```bash
   npm start
   ```

## Despliegue en Vercel

1. Crea una cuenta en [Vercel](https://vercel.com/).
2. Sube el código de la carpeta `frontend` a un nuevo proyecto.
3. Configura la variable de entorno `REACT_APP_API_URL` con la URL del backend desplegado (por ejemplo, en Railway).
4. Despliega el proyecto.
